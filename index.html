Welcome to Geemail.

Gee Mail is a secure console-version message exchanger, the project was divided into three parts: encryption and decryption, database, web parts.

We use passphrase to ensure the security of the encryption, and for the web parts, we implemented both php and cgi(an old school but interesting method.)Click on the github icon and see more about this project.
